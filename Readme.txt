Panduan Instalasi:
1. Pastikan Android Studio telah terinstall
2. lakukan clone repository
3. open project yang diclone tadi dengan android studio
4. tunggu build gradle selesai, lalu run dari android studio menggunakan device yang diinginkan

Cara Penggunaan:
- Pengguna memasukkan catatan pengeluaran dan pemasukan pada tab “Catatanku”. Dengan mengaktifkan lokasi pada Androidnya, aplikasi akan mendeteksi lokasi dan akan menentukan mata uang apa yang dipakai pengguna. Setiap pengguna menambahkan catatan keuangan pengguna akan mendapatkan koin. Koin tersebut akan berguna untuk membangun kota di Game Unity. 
- Pengguna bisa melihat riwayat transaksi di tab Transaksi.
- Pengguna bisa mendapatkan bonus koin dengan mengguncangkan Androidnya menggunakan fitur Shake to Win. Pengguna hanya bisa menggunakan fitur ini sekali dalam sehari.
